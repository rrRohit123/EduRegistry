# 🌿 EduRegistry - Student Management System


A modern Django CRUD project with secure login authentication.

## Setup

1. **Clone the repo** `git clone https://github.com/rrRohit123/EduRegistry.git`

2. **Create virtual environment** `python -m venv .venv`  
   `.venv\Scripts\activate`

3. **Install dependencies** `pip install -r requirements.txt`

4. **Run migrations** `python manage.py migrate`

5. **Create superuser** `python manage.py createsuperuser`

6. **Start server** `python manage.py runserver`

## Features

* **Login / Logout** - Secure staff authentication.
* **Add student** - Quick entry for new enrollment.
* **Edit student** - Update existing records easily.
* **Delete student** - Remove records with a confirmation prompt.
* **SQLite database** - Lightweight, local data storage.
